## Key Features

- **Yield Farming:** Stake tokens to earn rewards distributed from protocol fees.
- **Governance Token:** Enables decentralized decision-making and incentivizes participation.
- **Flash Loans:** Instant, uncollateralized loans that must be repaid within one transaction.
