- **Interest Rate Model:** Dynamic rates adjust based on supply and demand using an algorithmic model.
- **Liquidity Mining:** Incentivizes users to provide liquidity by distributing native tokens as rewards.
- **Governance Token:** Empowers community-driven protocol upgrades and parameter adjustments.
